This website uses Liquid Glass.

Liquid Glass is a design language developed by Apple as a unified visual theme for the graphical user interfaces for its suite of operating systems. It was announced on June 9, 2025, at the Worldwide Developers Conference (WWDC). Liquid Glass features a fluid, dynamic glass-like interface that reflects and refracts the background.


On non-Chromium browsers, a fallback blur will be used as using SVGs in backdrop-filter isn't supported.
